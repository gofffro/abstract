# abstract
Есть интерфейс абстрактной фабрики IGUIFactory, создающей кнопки и чекбоксы.

Конкретные фабрики создают объекты определённого стиля (восточный и западный).

Клиент получает фабрику и создаёт интерфейс из соответствующих объектов.

В Main создаются разные фабрики и демонстрируется их использование.

Паттерн Abstract Factory позволяет создавать семейства связанных объектов без привязки к конкретным классам.
